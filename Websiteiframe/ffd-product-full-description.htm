Description

Vinyl window sticker of a Soccer jersey with any name and number you choose.

Sticker can be customized with this information:

  • Name for jersey (optional; case-sensitive, 13 chars max.)
  • Number for jersey (optional; 1-2 digits)
  • Color of foreground (optional).  This is the foreground color only.  The background of the sticker is TRANSPARENT.  Be sure to consider the color of the surface to be applied to and make sure there's sufficient contrast!  For example, a dark green sticker would be barely visible on a dark-tinted window.

Sticker is new and is produced from 5-year exterior life vinyl.  It is 4 3/8" x 5 1/4" and comes with application instructions. 

This elegant design is ideal for external application on windows, walls, lockers, or any other clean dry surface.  There is no background; a dark target surface (such as a tinted window) is recommended for best results.
